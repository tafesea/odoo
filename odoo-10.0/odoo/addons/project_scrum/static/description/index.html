User guide

How to create a new Sprint?

  1. Go to Project > Search > Sprints
  2. To create a sprint, you need to specify the Start Date and End Date as well as a Scrum Team (see next point)

Note: The View Tasks button displays a list of the tasks in the sprint.

How to use Scrum Teams

To create a new Scrum Team, go to Project > Configuration > Scrum Teams or create one directly from the Sprint.

Scrum Teams can be used to have different sprint durations. For instance, the team #1 might have sprints that last 1 week while team #2 has sprints that last 3 days.
